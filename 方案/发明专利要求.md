提交中国发明专利申请时，**技术交底书**是发明人向专利代理机构提供的**最核心的技术资料**，是专利代理师撰写正式专利申请文件（包括说明书、权利要求书等）的基础。

技术交底书本身**不是正式的法律文件**，**不必严格符合**《专利法》要求的格式，但必须**清晰、完整、详细**地公开技术方案，以便专利代理师能充分理解并准确撰写。


## 核心保护 
在专利申请中，核心保护的正是“为解决某一技术问题而采取的所有技术手段和措施”

---

## 技术交底书应包含的主要内容

一份完整的技术交底书应充分公开技术方案，通常应包含以下几个关键部分：

### 1. 发明的基本信息
* **发明名称**：简单明了地体现发明的主题和类型（产品或方法）。
* **所属技术领域**：明确指出本发明技术方案所属或直接应用的技术领域。

### 2. 背景技术（现有技术状况）
* **现有技术的概述**：描述本发明涉及的现有技术现状。
* **最接近的现有技术**：描述发明人认为与本发明最接近的现有技术方案，并提供其信息来源（如专利公开号、非专利文献名称等）。
* **现有技术存在的问题或缺陷**：**实事求是**地指出最接近的现有技术中存在的问题、缺点或不足，这些问题是本发明要解决的。

### 3. 发明目的（要解决的技术问题）
* 针对现有技术中存在的缺陷，**明确指出**本发明要解决的**技术问题**（即发明想要达到的目的）。

### 4. 发明的技术方案
这是技术交底书的**核心部分**，需要**详细、完整**地公开本发明为解决上述技术问题所采用的**所有技术手段**。
* **技术方案描述**：按**步骤/层次**逻辑清楚地扩展描述（而不是仅按功能描述）。
* **对于产品发明**：应描述其结构组成，包括各部件的连接关系、相互作用等。
* **对于方法发明**：应说明所包括的**全部步骤**，操作、原料、工艺参数（如温度、压力、流量）及其**取值或选择范围**（给出较大的范围和优选范围）。

### 5. 具体实施方式（实施例）
提供**至少一个**或**多个**具体的、能够**实现本发明**技术方案的实施例。
* 描述如何应用技术方案，并给出必要的**参数、材料、步骤细节**。
* 提供**验证试验和试验数据**，证明本发明能够实现其目的并取得预期的**技术效果**。

### 6. 附图（如有必要）
如果技术方案涉及结构或流程，应提供附图以便理解。
* 附图可以是**结构图、流程图、示意图**等，以能清楚、完整地理解本专利为准。
* 图中**部件名称**只能用**编号**标出，部件名称在附图说明中说明。
* 附图线条应清晰、均匀，**不能标注尺寸**。

### 7. 本发明的关键点和欲保护点（可选）
如果发明人认为有必要，可以列举出本发明的**创新点、区别点**，以及希望得到保护的要点。这有助于专利代理师在撰写权利要求书时进行充分挖掘。

## 撰写技术交底书的要点总结

| 要点 | 说明 |
| :--- | :--- |
| **充分公开** | 必须将技术方案的所有细节**尽可能详细地写清楚**，使得**本领域技术人员**不需要付出创造性劳动就能实现该发明。 |
| **完整性** | 完整描述发明的所有技术特征，避免遗漏关键步骤或组件。 |
| **技术性** | 重点描述**技术方案本身**，而不是其商业价值或功能介绍。 |
| **客观对比** | 客观描述现有技术的缺点，并用本发明的技术方案和效果数据来**呼应**解决这些问题。 |
| **尽早提交** | 重要的技术构思应在**向合作方公开或以其他方式公开之前**尽早准备和提交交底书，以便及时申请专利。 |

**重要提示：**

技术交底书是**技术文件**，正式的专利申请文件是**法律文件**，专利代理师的作用就是将您的技术交底书转化为符合专利法要求的法律文件。因此，您需要做的就是提供**最详尽、最准确**的技术资料。
